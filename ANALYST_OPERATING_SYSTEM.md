# Analyst Operating System

## Core Rule
Every number must be explainable.

## Analyst Workflow

1. Understand the business question
2. Define the metric clearly
3. Validate the data source
4. Run analysis (SQL / Python)
5. Check for errors or bias
6. Interpret results
7. Recommend action

## Common Failure Points
- Wrong joins
- Wrong grain (MOST COMMON)
- Missing filters
- Misaligned definitions
- Double counting

## Senior Analyst Thinking
If stakeholders disagree on numbers:
→ The data model is broken, not the people.
